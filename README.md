# Indie Game Publisher Recommender
[Live App](https://indie-publisher-search.streamlit.app/)

A content-based recommender system designed to help indie developers identify potential publishers through historical catalog similarity.

The project matches game concepts to publishers by:
- Vectorizing Steam Tags: Analyzing the multidimensional relationship between game genres, sub-genres, and mechanics.
- Price Point Alignment: Filtering results based on target price brackets to ensure commercial compatibility.
- Evidence-Based Discovery: Surfacing concrete examples of previously published titles to provide context for outreach.

The purpose of this project is to bridge the gap between creative development and business strategy. By quantifying "publisher fit" through historical data, indie developers can move beyond guesswork and approach publishers who have a proven track record with similar titles.
